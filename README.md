## 프로젝트 참여자

- 기획 : 아트 개발 공동
- 개발 : 박태호 , 오광원
- 아트 : 박민지 , 조은일

## 사용프로그램
- Unity
- GitHub : 코드 저장용도 
- SourceTree : 버전관리 용이 /공동작업 가능
- Visual Studio 2015
